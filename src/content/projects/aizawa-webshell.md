Aizawa Webshell is a curated set of lightweight PHP webshells that pair with the Aizawa client. They are intentionally minimal so researchers can study payload behavior and tweak them for controlled red-team exercises.

## What it does
- Provides multiple PHP webshell variants that speak Aizawa's header-based protocol.
- Serves as a teaching tool for payload design and hardening tests.
- Keeps deployments small so they can be dropped into tight lab environments.

## Highlights
- Compatible with the Aizawa client out of the box.
- Built for demonstrations of security controls and bypass research.
- Simple, readable payloads to encourage experimentation.
